# Diablo III baseline source library

Purpose: clean source-move inventory for later chaos-draft/deckbuilding work. This file is not implementation code yet. It is a design-reference list of Diablo III source mechanics, grouped so they can later become cards, items, potions, upgrades, passives, resource modules, or starting kit modules.

Scope: baseline/original Diablo III classes discussed so far: Barbarian, Demon Hunter, Monk, Witch Doctor, and Wizard. Crusader, Necromancer, legendary items, set bonuses, followers, gems, paragon, and Reaper of Souls-only level-61 skills are intentionally parked outside the baseline section for now.

Tag legend used below:

- `core-kit`: can define Square/Triangle/basic/heavy identity.
- `active-card`: can become an LB/RB castable card.
- `generator`: generates class resource.
- `spender`: spends class resource.
- `passive`: always-on rule/item candidate.
- `resource`: meter or charge system.
- `summon`: pet/turret/decoy/ally.
- `mobility`: dash, leap, teleport, tumble, charge, movement.
- `control`: stun, fear, slow, blind, freeze, charm, pull, knockback.
- `zone`: persistent ground or area effect.
- `channel`: held/cycling effect.
- `upgrade`: rune/variant candidate.
- `potion`: temporary single-use effect candidate.

## Baseline class resources

| Class | Source resource | Hades/OSFE chaos interpretation | Tags |
|---|---|---|---|
| Barbarian | Fury. Generated by attacks/shouts/impact actions; spent on heavy attacks and rage abilities; can decay depending on implementation era. | Aggression meter for melee/bruiser cards. | resource, fury |
| Demon Hunter | Hatred + Discipline. Hatred fuels attacks and regenerates quickly; Discipline fuels defensive/acrobatic/shadow tools and regenerates slowly. | Two-resource split: offense pool + defense/mobility pool. | resource, hatred, discipline |
| Monk | Spirit. Generated by primary attacks; spent on techniques, mantras, defensive and focus skills. | Combo/spirit pool for martial kit. | resource, spirit |
| Witch Doctor | Mana. Large pool, slower regeneration, many summons/DoT/control skills. | Big magic pool with pet/curse/zone cards. | resource, mana |
| Wizard | Arcane Power. Smaller, fast-regenerating pool; powers spells, armor, beams, summons. | Fast-cycling spell resource. | resource, arcane-power |
| Charges | Some Diablo skills store charges instead of normal resource costs. | Independent cooldown/stock counters. | resource, charge |
| Cooldowns | Many defensive/ultimate skills are time-gated. | Card cooldown or temporary exhaust rule. | resource, cooldown |

## Notes for later Slay-the-Spire-style structure

- Cards: active skills, Signature spells, spenders, summons, shouts, mobility moves, channels, zones.
- Items/relics: passives, armor/mantra-style persistent effects, weapon-tag bonuses.
- Potions: long-cooldown ultimates, one-use summons, emergency defenses.
- Upgrades: runes become upgrade branches or variant cards.
- Resources: each class resource can be drafted as a resource module, allowing weird cross-class dead cards or missing-resource failures.
- Characters/starting kits: a starting kit could randomly draft one class resource, one core-kit/generator, two active cards, and one passive/item.
